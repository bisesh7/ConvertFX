# Next.js Currency Converter

A dynamic web application built using Next.js and TypeScript, this Currency Converter allows real-time conversion across a variety of global currencies with a seamless responsive design using React Bootstrap. Ideal for users needing quick and accurate currency conversions on both mobile and desktop platforms.

## Features

- **Real-Time Conversion**: Instantaneously convert amounts between various currencies without the need for a reload or button click.
- **Responsive Layout**: Utilizes React Bootstrap to ensure that the app looks great on all devices from phones to desktops.
- **User-Friendly Interface**: Designed with simplicity and usability in mind to provide users a straightforward and intuitive experience.
